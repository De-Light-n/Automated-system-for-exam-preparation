# Automated System for Exam Preparation

Автоматизована система для підготовки до іспитів з аналізом лабораторних робіт та персоналізованими рекомендаціями.

## Основні функції

### 1. Аналіз виконаних лабораторних робіт

- Перевірка правильності рішень
- Виявлення типових помилок
- Оцінка рівня засвоєння теми

### 2. Формування узагальненої статистики

- Відсоток правильно виконаних завдань
- Порівняння результатів між різними лабораторними
- Визначення прогресу студента

### 3. Генерація рекомендацій для підготовки

- Підсвітка тем, які вимагають додаткового опрацювання
- Створення списку питань для самоперевірки
- Формування персонального плану підготовки

### 4. Екзаменаційний тренажер

- Автоматичне створення тестових запитань
- Режим симуляції екзамену з обмеженням часу
- Підрахунок балів та оцінка готовності

### 5. Візуалізація результатів

- Графіки успішності
- Таблиці з розбиттям по темах
- Індикатори рівня підготовки

## Технології

- **Backend**: FastAPI (Python)
- **Frontend**: React
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy

## Структура проекту

```
├── backend/          # FastAPI backend
├── frontend/         # React frontend
└── docs/            # Документація
```

## Запуск проекту

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
