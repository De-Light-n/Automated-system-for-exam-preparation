# Assets Directory

## Structure

```
assets/
├── images/          # Screenshots and feature previews
│   ├── flashcards-preview.png
│   ├── quiz-preview.png
│   ├── mindmap-preview.png
│   └── chat-preview.png
└── avatars/         # Default user avatars
    └── default-avatar.png
```

## Image Guidelines

### Feature Screenshots (images/)

- **Format**: PNG or WebP
- **Size**: 1200x800px (3:2 aspect ratio)
- **Quality**: High quality, optimized for web
- **Content**: Clear demonstration of the feature in action

### Avatars (avatars/)

- **Format**: PNG with transparency
- **Size**: 256x256px (square)
- **Style**: Simple, recognizable icons
