# Швидкий старт

## 1. Backend (FastAPI)

### Встановлення та запуск:

```powershell
# Перейти в папку backend
cd backend

# Створити віртуальне середовище
python -m venv venv
.\venv\Scripts\Activate.ps1

# Встановити залежності
pip install -r requirements.txt

# Налаштувати .env файл
Copy-Item .env.example .env
# Відредагуйте .env файл з правильними налаштуваннями БД

# Запустити сервер
uvicorn app.main:app --reload
```

Backend буде доступний на: http://localhost:8000
API документація: http://localhost:8000/docs

## 2. Frontend (React)

### Встановлення та запуск:

```powershell
# Відкрийте новий термінал і перейдіть в папку frontend
cd frontend

# Встановити залежності
npm install

# Запустити dev сервер
npm start
```

Frontend буде доступний на: http://localhost:3000

## 3. База даних

Потрібно створити PostgreSQL базу даних:

```sql
CREATE DATABASE exam_prep_db;
```

## Наступні кроки для розробки:

1. **Backend:**

   - Налаштувати Alembic для міграцій БД
   - Реалізувати сервіси для кожного endpoint
   - Додати автентифікацію JWT
   - Реалізувати алгоритми аналізу лабораторних робіт
   - Додати генерацію питань для екзамену

2. **Frontend:**

   - Підключити реальні API запити
   - Додати обробку помилок
   - Реалізувати захист роутів (приватні маршрути)
   - Покращити UI/UX
   - Додати більше графіків та візуалізацій

3. **Додаткові функції:**
   - Завантаження файлів лабораторних робіт
   - Експорт статистики
   - Налаштування таймерів для екзамену
   - Email нотифікації
